# Generative AI Learning Lab

Repository focused on learning and experimenting with **Generative AI**, **LLMs** and **RAG**, through practical notebooks and small applied projects.

## Overview

This repository contains:
- Notebooks based on training materials and adapted to new datasets
- Experiments with LLMs and generative AI techniques
- Small projects oriented to real-world use cases
- Reproducible and structured learning content

## Tech Stack

- Python
- Jupyter Notebooks
- Pandas / NumPy
- LLM frameworks (OpenAI, LangChain, etc.)
- SQL
- Data processing & automation tools
